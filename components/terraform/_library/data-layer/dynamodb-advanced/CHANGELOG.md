# Changelog

## [1.0.0] - 2025-12-02

### Added
- Initial release of DynamoDB Advanced module
- Support for on-demand and provisioned billing modes
- Auto-scaling for provisioned capacity
- Global tables for multi-region replication
- DynamoDB Streams integration
- Point-in-time recovery
- KMS encryption
- TTL support
- GSI and LSI configuration
- Deletion protection
- Standard and Infrequent Access table classes
